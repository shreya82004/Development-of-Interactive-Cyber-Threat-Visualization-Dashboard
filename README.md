## 📌 Project Overview  

CyberShield is a real-time **Security Operations Center (SOC) Dashboard** built using React.  
It simulates a live cyber threat intelligence monitoring platform with dynamic visualizations and interactive security tools.

The dashboard refreshes automatically every **2.8 seconds**, simulating continuous global cyber attack activity.

This project demonstrates how modern SOC systems monitor, analyze, and visualize cybersecurity threats in real time.

---

## 🎯 Objectives  

- Simulate real-time cyber threat monitoring  
- Visualize attack severity and global distribution  
- Demonstrate SOC workflow concepts  
- Implement interactive security tools  
- Showcase frontend-driven cybersecurity analytics  

---

### 1️⃣ Dashboard
